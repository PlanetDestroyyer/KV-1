## How to Use This Curriculum

### Strategy 1: Sequential Learning (Recommended)
Ask questions in order, letting the system build knowledge progressively:
```bash
# Phase 1 (Day 1-3)
python run_self_discovery.py "What is addition and how does it work?"
python run_self_discovery.py "What is multiplication and how does it relate to addition?"
# ... continue through Phase 1

# Phase 2 (Day 4-7)
python run_self_discovery.py "What is a limit and why is it important?"
# ... continue
```

### Strategy 2: Batch Learning (Faster)
Create a script to ask multiple questions automatically:
```bash
# Create batch file
cat > batch_questions.txt <<EOF
What is addition and how does it work?
What is multiplication and how does it relate to addition?
What are prime numbers and why are they important?
...
EOF

# Run batch learning
while read question; do
    python run_self_discovery.py "$question" --max-attempts 3
    sleep 5  # Avoid rate limits
done < batch_questions.txt
```
